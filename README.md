# Bootcamp_AssistenteDeVoz

**Desafio 02 do Bootcamp do Bradesco com a DIO**
criação de uma assistente de voz que captura áudio do microfone, transcreve usando a IA Whisper, processa a resposta via GPT-4 e responde com voz sintetizada.

**Linguagens:** Python e JavaScript

Caso deseja replicar esse projeto, as bibliotecas instaladas foram:
- pip install sounddevice
- pip install scipy 
- pip install openai-whisper 
- pip install openai 
- pip install gTTS 
- pip install pygame 
- pip install setuptools-rust 
- pip install python-dotenv
- choco install ffmpeg
